# Example: Multi-Step Reasoning Stress Test

### Prompt
اشرح كيف تعمل خوارزمية Merge Sort عبر ثلاث مراحل: التقسيم – المعالجة – الدمج، ثم وضّح كيف يؤثر استخدام الذاكرة الإضافية على سرعتها.

### Ideal Answer
1. **التقسيم:** يتم تقسيم المصفوفة...
2. **المعالجة:** يتم ترتيب كل جزء...
3. **الدمج:** يتم دمج الأجزاء...
4. **الذاكرة:** Merge Sort يحتاج ذاكرة إضافية...
